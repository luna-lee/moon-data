# moon-dv-echarts-group

-   渲染echarts组件组，flex布局！
-   [echarts 文档](https://echarts.apache.org/zh/index.html)

### Props

| 参数        | 说明                      | 类型 | 可选值 | 默认值 |
| ----------- | ------------------------- | ---- | ------ | ------ |
| options      | 多个echarts 的 option 配置数组    | Array   | -      | -      |

### Slot

| 参数    | 说明 |
| ------- | ---- |
| default | echarts组前一个元素 |
| last | echarts组后一个元素 |
