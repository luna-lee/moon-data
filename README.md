## 介绍

-   #### 基于 vue2 的可视化页面设计工具，所见即所得。
-   #### 适用于大型大屏项目，文档管理项目，报表项目，能胜任各种复杂的交互页面。
-   #### 增加工作空间，适合多人协同开发复杂业务页面。
-   #### 项目亦组件，组件亦项目。
